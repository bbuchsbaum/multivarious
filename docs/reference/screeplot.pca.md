# Screeplot for PCA

Displays the variance explained by each principal component as a bar or
line plot.

## Usage

``` r
# S3 method for class 'pca'
screeplot(x, type = "barplot", main = "Screeplot", ...)
```

## Arguments

- x:

  A `pca` object.

- type:

  "barplot" or "lines".

- main:

  Plot title.

- ...:

  Additional args to pass to base R plotting.
