# Extract the Block Indices from a Multiblock Projector

Extract the Block Indices from a Multiblock Projector

## Usage

``` r
# S3 method for class 'multiblock_projector'
block_indices(x, i, ...)
```

## Arguments

- x:

  A `multiblock_projector` object.

- i:

  Ignored.

- ...:

  Ignored.

## Value

The list of block indices.
