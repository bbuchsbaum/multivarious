# center and scale each vector of a matrix

center and scale each vector of a matrix

## Usage

``` r
standardize(preproc = prepper(), cmeans = NULL, sds = NULL)
```

## Arguments

- preproc:

  the pre-processing pipeline

- cmeans:

  an optional vector of column means

- sds:

  an optional vector of sds

## Value

a `prepper` list
