# Extract coefficients from a cross_projector object

Extract coefficients from a cross_projector object

## Usage

``` r
# S3 method for class 'cross_projector'
coef(object, source = c("X", "Y"), ...)
```

## Arguments

- object:

  the model fit

- source:

  the source of the data (X or Y block), either "X" or "Y"

- ...:

  extra args

## Value

the coefficients
