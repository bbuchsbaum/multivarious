# Shared PCA basis specification

Shared PCA basis specification

## Usage

``` r
shared_pca(ncomp, fit_on = c("nuisance_residual", "full", "whitened_residual"))
```

## Arguments

- ncomp:

  Number of shared basis components.

- fit_on:

  Basis fitting target.

## Value

A basis specification for a shared PCA basis.
