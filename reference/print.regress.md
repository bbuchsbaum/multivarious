# Pretty Print Method for `regress` Objects

Display a human-readable summary of a `regress` object using crayon
formatting, including information about the method and dimensions.

## Usage

``` r
# S3 method for class 'regress'
print(x, ...)
```

## Arguments

- x:

  A `regress` object (a bi_projector with regression info).

- ...:

  Additional arguments passed to
  [`print()`](https://rdrr.io/r/base/print.html).
