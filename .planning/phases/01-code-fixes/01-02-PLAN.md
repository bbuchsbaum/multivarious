  <files>R/plsc_inference.R</files>
  <action>
The generic `bootstrap(x, nboot, ...)` has a specific signature, but `bootstrap.plsc(x, X, Y, nboot, ...)` does not match (has X, Y before nboot).

Modify `bootstrap.plsc` in R/plsc_inference.R (around line 288) to match the generic signature:

```r
#' @export
bootstrap.plsc <- function(x, nboot = 500, ...) {
  # Extract X and Y from ...
  args <- list(...)
  X <- args$X
  Y <- args$Y

  if (is.null(X) || is.null(Y)) {
    stop("bootstrap.plsc requires X and Y to be passed as named arguments via ...", call. = FALSE)
  }

  bootstrap_plsc(x, X = X, Y = Y, nboot = nboot, ...)
}
```

This maintains backward compatibility by accepting X and Y through `...` while matching the generic signature.
