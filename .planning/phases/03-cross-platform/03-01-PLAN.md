Run the following R commands from the package directory to submit to all three build services:

```r

# Submit to macOS M1 (returns URL)

```

These commands:
1. Build the source package (multivarious_0.3.0.tar.gz)
2. Upload to the respective build services
3. Win-builder: Results sent to maintainer email (bbuchanan@rotman.baycrest.org based on DESCRIPTION)
4. Mac-builder: Results available at returned URL

Capture and report:
- Confirmation messages from each submission
- The mac-builder URL returned by check_mac_release()

Note: Submissions complete quickly (~1 minute each), but results take ~30 minutes to arrive.
