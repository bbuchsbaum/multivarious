## Key Dependencies

**Critical (core functionality):**
- `Matrix` - Sparse matrix support and linear algebra operations
- `rlang` - Modern R programming patterns, error handling
- `chk` - Input validation
- `lifecycle` - API deprecation management

**SVD/Eigenvalue Computation:**
- `corpcor` - fast.svd for efficient SVD (`R/svd.R`)
- `rsvd` - Randomized SVD (`R/svd.R`)
- `irlba` - Truncated SVD for large matrices (`R/svd.R`)
- `RSpectra` - Spectral decomposition, svds (`R/svd.R`, `R/geneig.R`)
- `svd` - propack.svd (`R/svd.R`)
- `geigen` - Generalized eigenvalue problems (`R/geneig.R`)
- `PRIMME` - Iterative eigensolver for large symmetric problems (`R/geneig.R`)

**Statistical Methods:**
- `glmnet` - Regularized regression (`R/regress.R`)
- `pls` - Partial Least Squares (`R/plsc.R`)
- `MASS` - Statistical methods
- `GPArotation` - Factor rotation methods (`R/pca.R`)
- `matrixStats` - Fast matrix statistics

**Visualization:**
- `ggplot2` - Plotting framework
- `ggrepel` - Non-overlapping text labels

**Data Manipulation:**
- `tibble` - Modern data frames
- `dplyr` - Data manipulation
- `tidyr` - Data tidying

**Parallel Computing:**
- `future` - Parallel backend abstraction
- `future.apply` - Parallel apply functions

**Utilities:**
- `proxy` - Distance/similarity computations
- `cli` - Command line interface utilities
- `crayon` - Terminal text formatting
- `withr` - Temporary state management
- `assertthat` - Assertions

**Suggested (optional):**
