## Directory Purposes

**R/:**
- Purpose: All R source code for the package
- Contains: 23 `.R` files implementing projector classes, algorithms, and utilities
- Key files: `projector.R`, `bi_projector.R`, `all_generic.R`, `pre_process.R`

**man/:**
- Purpose: Generated documentation files
- Contains: `.Rd` files auto-generated by roxygen2 from source comments
- Key files: One file per documented function/class
- Generated: Yes, by `devtools::document()`
- Committed: Yes

**tests/testthat/:**
- Purpose: Unit tests using testthat framework
- Contains: 17 test files covering core functionality
- Key files: `test_pca.R`, `test_projector.R`, `test_preprocess.R`

**vignettes/:**
- Purpose: Long-form tutorials and documentation
- Contains: 14 `.Rmd` vignettes covering various use cases
- Key files: `Introduction.Rmd`, `SVD_PCA.Rmd`, `PreProcessing.Rmd`

**docs/:**
- Purpose: pkgdown-generated website
- Contains: HTML documentation, articles, reference pages
- Generated: Yes, by `pkgdown::build_site()`
- Committed: Yes

**experimental/:**
- Purpose: Work-in-progress code not yet part of main package
- Contains: Experimental implementations
- Committed: Yes, but not exported

## Key File Locations

**Entry Points:**
- `R/pca.R`: Primary PCA implementation - `pca()`
- `R/svd.R`: SVD wrapper - `svd_wrapper()`
- `R/plsc.R`: Partial Least Squares Correlation - `plsc()`
- `R/regress.R`: Regression-based projection - `regress()`

**Core Classes:**
- `R/projector.R`: Base `projector` class
- `R/bi_projector.R`: Two-way `bi_projector` class
- `R/twoway_projector.R`: Cross-domain `cross_projector` class
- `R/composed_projector.R`: Sequential `composed_projector` class
- `R/multiblock.R`: Multiblock projector classes

**Generic Functions:**
- `R/all_generic.R`: 50+ S3 generic definitions (project, reconstruct, scores, etc.)
- `R/preproc-generics.R`: Preprocessing generics (fit, transform, inverse_transform)

**Preprocessing:**
- `R/pre_process.R`: Preprocessing pipeline and steps (center, colscale, standardize)
- `R/preproc-utils.R`: Preprocessing utilities (is_fitted, mark_fitted, check_fitted)

**Configuration:**
- `DESCRIPTION`: Package dependencies, version, metadata
- `NAMESPACE`: Exported functions (auto-generated)
- `_pkgdown.yml`: Website structure configuration

**Testing:**
- `tests/testthat.R`: Test runner setup
- `tests/testthat/test_*.R`: Individual test files

## Naming Conventions

**Files:**
- Source files: `snake_case.R` (e.g., `pre_process.R`, `bi_projector.R`)
- Test files: `test_<component>.R` (e.g., `test_pca.R`)
- Vignettes: `PascalCase.Rmd` or `snake_case.Rmd`

**Functions:**
- Constructors: `snake_case()` (e.g., `projector()`, `bi_projector()`)
- Generics: `snake_case()` (e.g., `project()`, `partial_project()`)
- Internal helpers: `.prefixed_snake_case()` (e.g., `.cache_inv()`)
- S3 methods: `generic.class()` (e.g., `project.projector()`)

**Classes:**
- S3 classes: `snake_case` (e.g., `projector`, `bi_projector`, `cross_projector`)
- Compound classes: `adjective_noun` (e.g., `composed_projector`, `partial_projector`)

**Parameters:**
- Arguments: `snake_case` (e.g., `new_data`, `colind`, `ncomp`)
- Internal variables: `snake_case` (e.g., `nd_proc`, `inv_proj`)

## Where to Add New Code

**New Dimensionality Reduction Algorithm:**
- Primary code: `R/<algorithm_name>.R`
- Constructor function returning `bi_projector` or `cross_projector`
- Tests: `tests/testthat/test_<algorithm_name>.R`
- Vignette: `vignettes/<AlgorithmName>.Rmd`

**New Projector Type:**
- Implementation: `R/<type>_projector.R`
- Inherit from `projector`, `bi_projector`, or `cross_projector`
- Implement required S3 methods in same file
- Tests: `tests/testthat/test_<type>_projector.R`

**New S3 Generic:**
- Add generic definition to `R/all_generic.R`
- Add `@export` roxygen tag
- Implement methods in relevant class files
- Update `NAMESPACE` via `devtools::document()`

**New Preprocessing Step:**
- Add function to `R/pre_process.R`
- Follow pattern: return `prepper` with `forward`, `apply`, `reverse` methods
- Example: Copy structure from `center()` or `colscale()`

**New Utility Function:**
- Shared utilities: `R/utils.R`
- Preprocessing utilities: `R/preproc-utils.R`
- Internal helpers: Use `.` prefix, add `@keywords internal` and `@noRd`
